Background

FontFaceSet is an interface defined as part of the CSS Font Loading API. It's used to load font faces, and check the status of previously requested fonts. The FontFaceSet interface for the current HTML page is available in JavaScript as document.fonts.

This sample shows equivalent techniques for defining a new FontFace and adding it to the page's FontFaceSet. One technique is to use the FontFaceSet.add() method via JavaScript, and the other technique is to use CSS's @font-face rule.

The sample also shows alternative approaches to the initial display of text before the custom fonts are loaded. In one case, the text is displayed unconditionally, but will start out being displayed using the page's default font until the custom FontFace loads. In the other case, the text is styled with visibility: hidden initially, and then after the document.fonts.ready Promise resolves, the text is displayed.
