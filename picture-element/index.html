<picture> Element Sample

Available in Chrome 38+

The <picture> element informs Chrome which version of the picture to fetch, based on some criteria, like user's viewport, screen type etc.

In the example below, the browser looks for the first <source> element where media query matches user's current setup, and fetches the image specified in the srcset attribute of that <source> element.

<img> element is used as a fallback for browsers not supporting the <picture> element.

Resize the browser to see different versions of the picture loading at different viewport sizes.
